WhatsApp Message System

To use the WhatsApp Message System component on your website, follow these steps:

  1. Download the JavaScript file whatsappMessageSystem.js.
  2. Include the JavaScript file in your HTML page by adding the following line before the closing </body> tag:
    3. <script src="whatsappMessageSystem.js"></script>
  3. Replace YOUR_WHATSAPP_NUMBER in the JavaScript file with your actual WhatsApp number (in international format, without the + sign).
  4. Open your HTML page in a browser to see the WhatsApp icon on the bottom-right corner. Click the icon to open the popup form.

Example HTML structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>WhatsApp Message System</title>
</head>
<body>
    <!-- Your page content -->

    <script src="whatsappMessageSystem.js"></script>
</body>
</html>
